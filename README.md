Pokédex
Este é um projeto simples de uma Pokedex desenvolvido com HTML, CSS e JavaScript. A Pokedex permite que você pesquise e visualize informações sobre os Pokémon, como nome, tipo, e imagem.

Funcionalidades
Busca por Pokémon : Encontre um Pokémon pelo nome e veja suas informações feitas.
Exibição de Imagem : Cada Pokémon possui sua imagem oficial.
Informações do Pokémon : Exibe nome, tipo e outras informações básicas.
Tecnologias Utilizadas
HTML : Para a estrutura da página.
Angular:
CSS : Para estilização da página.
JavaScript : Para a lógica de interação e manipulação de dados da Pokedex.
API : A Pokedex usa uma API pública do Pokémon (PokéAPI) para obter as informações dos Pokémon.
